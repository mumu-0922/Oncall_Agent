# Open-source reference workspace

这个目录只用于隔离拉取外部开源项目，作为架构/实现参考，不作为本项目源码依赖。

硬约束：

1. 不从 `app/`、`mcp_servers/`、`tests/` import 这里的代码。
2. 不直接复制大段实现；需要借鉴时先写设计说明，再用本项目风格重写。
3. 若确需引入少量代码片段，必须记录来源、许可证、文件路径和修改点。
4. 该目录应被 `.gitignore` 忽略，避免把第三方仓库提交进主仓库。
5. 参考项目更新只用 shallow clone / pull，不改第三方仓库源码；魔改产物放回本项目独立模块。
